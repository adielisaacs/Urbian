Toilet Talking 
-------------
1. Mysql AND (Xampp OR Apache) is needed to run this application
2. Create a database connection.
3. Copy the db.sql file in the \Urbian\includes folder.
4. Run the script in Mysql
5. Copy the connection details and UPDATE the dbconnection.php in the following directories
	\Urbian\includes
	\Urbian\admin\models

6. Deploy the Application/Folder structure to XAMPP OR Apache.
7. Start server

8. Access the application on http:\\server:xxxx\Urbian
9. Access the application Admin console on http:\\server:xxxx\Urbian\admin
